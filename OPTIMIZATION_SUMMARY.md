## 已知限制

1. **评论存储**：当前评论存储在 localStorage，刷新后会丢失。可通过 `/api/feed-comment` 保存到数据库。

2. **点赞同步**：点赞数在本地更新后持久化到 Supabase，但跨设备同步需要手动调用 `/api/feed-like`。

3. **AI 摘要生成**：每次最多并发 5 条，避免 API 过载。对于大量新 feed，生成可能较慢。

4. **冷启动体验**：依赖于 SQL 种子数据的执行。如果未执行种子数据，圈子会为空。
