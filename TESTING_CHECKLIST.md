# UniPass 冷启动优化 - 测试清单

## 快速测试指南

### 部署前必做

1. **数据库迁移**（必须按顺序执行）
   ```sql
   -- 1. 升级 feeds 表结构
   执行: supabase/sql/feeds_schema_upgrade.sql
   
   -- 2. 创建游客数据表
   执行: supabase/sql/app_guest_data.sql
   
   -- 3. 添加 10 城市种子数据
   执行: supabase/sql/feeds_seed_cq_hz_sz_bj.sql
   
   -- 4. 添加大厂加强数据
   执行: supabase/sql/feeds_seed_bigtech_boost.sql
   ```

2. **验证环境变量** (`.env.local`)
   ```
   NEXT_PUBLIC_SUPABASE_URL=...
   NEXT_PUBLIC_SUPABASE_ANON_KEY=...
   OPENAI_API_KEY=...
   OPENAI_BASE_URL=...
   AI_MODEL=...
   ```

3. **运行项目**
   ```bash
   npm run dev
   ```

## 已知问题和解决方案

### 1. 圈子为空
**原因**: 种子数据未执行
**解决**: 执行 SQL 文件中的 INSERT 语句

### 2. AI 摘要显示为默认文本
**原因**: AI API 调用失败或未配置
**解决**: 检查 .env.local 中的 API Key

### 3. 评论发布后立即消失
**原因**: 评论只保存在 localStorage，未持久化到数据库
**解决**: 调用 `/api/feed-comment` 保存

### 4. 同城筛选不工作
**原因**: 用户位置未检测
**解决**: 授予浏览器地理位置权限
